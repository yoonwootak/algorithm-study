# algorithm-study

코딩테스트 준비 과정에서 푼 문제들을 기록하는 저장소입니다.

---

## 프로젝트 개요

- 형태: 개인 학습 기록용 저장소
- 대상: `Baekjoon Online Judge`
- 사용 언어: `Java`
- 핵심 목적: 문제 풀이 기록, 풀이 경험 축적, 유형 적응

## 프로젝트 소개

- 현재는 백준 문제 풀이를 중심으로 기록하고 있습니다.
- 문제별 폴더를 만들고, 각 폴더 안에 `Main.java` 형태로 풀이 코드를 저장하고 있습니다.
- 코딩테스트 준비 초반이라 지금은 풀이 설명을 자세히 정리하기보다, 문제를 더 많이 풀면서 풀이 경험을 쌓는 데 집중하고 있습니다.

## 정리 방식

- `Baekjoon Online Judge` 아래에 문제 이름 기준으로 폴더를 생성합니다.
- 각 문제 폴더 안에 `Main.java` 파일로 풀이 코드를 기록합니다.

---

## 목적

- 코딩테스트 문제 풀이 기록
- 자주 나오는 유형에 익숙해지기
- 꾸준히 문제를 풀며 풀이 감각 쌓기
